Bienvenida

En la primera clase se introduce algunos conceptos basicos. Comenzando desde cero vas a aprender a crear y ejecutar programas basicos. Tambien presentamos algunos tipos de datos primitivos como los enteros, flotantes, booleanos, strings, funciones y variables. Mas adelante en el curso aprenderas lo necesario para crear programas mas avanzados. Comenzaremos por el tradicional programa 'Hola Mundo', que es un programa que muestra 'Hola Mundo' al ejecutarse.
  1. Apreta el boton amarillo que dice 'Run!' para ejecutar el programa.
  2. Apreta el boton amarillo que dice 'Next' para avanzar de nivel.
