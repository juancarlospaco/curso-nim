Orientado a Objetos

Se puede extender el codigo del ejemplo del 'Gatito', para implementar Orientacion a Objetos.

Para heredar se usa el operador 'of', con la sintaxis "A of B", es decir 'ref object of RootObj', 'RootObj' es madre de todos los objetos, 'Gatito' hereda de 'Animal', usando el operador 'of', con la sintaxis 'ref object of Animal'. Gatito tiene atributos y funciones de 'Animal'.

type Animal = ref object of RootObj # Animal es el objeto base. edad: int nombre: string type Gatito = ref object of Animal # Gatito hereda de Animal. durmiendo: bool proc renombrar(self: var Animal, nombre: string) = self.nombre = nombre proc maullar(self: Gatito) = echo "Miau Miau" proc crecer(self: var Gatito) = inc self.edad proc dormir(self: var Gatito) = self.durmiendo = true proc despertar(self: var Gatito) = self.durmiendo = false var gato = Gatito(edad: 1, nombre: "zoe", durmiendo: false) gato.maullar() gato.crecer() gato.dormir() gato.despertar()
