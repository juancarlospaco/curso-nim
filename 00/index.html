echo

La funcion 'echo' sirve para mostrar por pantalla cualquier valor en una representacion de 'string', 'echo' convertira los valores a 'string' y los imprimira en la linea de comando, puede tomar cualquier cantidad de argumentos de cualquier tipo, pero debe tener por lo menos 1 argumento, la representacion de 'string' es legible por humanos.

Se observa que existen distintos "Tipos de Datos", esto es por que una computadora no puede distinguir los datos, no puede distinguir cuando te refieres a un tres de la palabra "Tres", o cuando te refieres a un tres del numero "3", o cuando te refieres a un tres del numero con coma flotante "3.0".

echo 1, true, 2.0, "OwO", false, -9, 0

Booleanos

El tipo booleano solo puede tener 2 valores, verdadero que es 'true' o falso que es 'false'. Se puede invertir el valor usando 'not', por ejemplo 'not true' es 'false' y 'not false' es 'true'. Cuando se crea un booleano su valor por defecto es 'false'.

Tambien existe el operador 'and' que cumple la funcion de "Y", es decir para juntar dos expresiones booleanas en una sola, ejemplo 'true and true' es 'true', 'true and false' es 'false'.

Asi mismo el operador 'or' que cumple la funcion de "O", es decir para juntar dos expresiones booleanas en una sola, ejemplo 'true or true' es 'true', 'true or false' es 'true'.

  1. Apreta el boton amarillo que dice 'Run!' para ejecutar el programa.
  2. Apreta el boton amarillo que dice 'Next' para avanzar de nivel.
echo true echo false echo not true echo not false echo true and true echo true and false echo true or true echo true or false

Enteros

Los numeros enteros llamados 'int' (integer) son numeros negativos o positivos, incluyendo el cero. En el caso de los numeros negativos el menos debe estar pegado al numero, es decir '- 5' debe ser '-5'. Cuando se crea un entero su valor por defecto es '0'.
  1. Apreta el boton amarillo que dice 'Run!' para ejecutar el programa.
echo 0 echo 1 echo 9 echo -9 echo -5 echo -0 # -0 es 0

Strings

El texto o cadenas de texto o cadenas de caracteres se llaman 'string', y puede contener cualquier texto como palabras, numeros, puntuacion, etc. Cuando se crea un string su valor por defecto es '""', que es un string vacio. Se puede concatenar dos o mas 'string' usando '&'. 'string' de multiples lineas o multilinea se crean usando triple comillas.echo "Hola" echo "1234" echo "true" echo "" # String vacio

Flotantes

Los numeros de punto flotante o numeros de coma flotante se llaman 'float', pueden ser negativos, positivos, cero, e infinito. Cuando se crea un flotante su valor por defecto es '0.0'.

'float' de acuerdo con el estandard de numeros de punto flotante tiene unos valores especiales, que indican infinito negativo e infinito positivo, que son numeros tan grandes que la PC no los puede interpretar, tambien una forma de No-Numero (Not A Number), estos "valores especiales" no se trataran en detalle en este curso.

echo 2.0 echo 3.14 echo 99.50

Matematica

Matematica basica se puede realizar con distintos operadores y valores numericos.
  • Sumar usa el operador '+'.
  • Restar usa el operador '-'.
  • Multiplicar usa el operador '*'.
  • Division flotante usa el operador '/'.
  • Division entera usa el operador 'div'.
  • Modulo usa el operador 'mod'.
echo 1 + 2 echo 2 - 1 echo 2 * 2 echo 4 / 2 echo 4 div 2 echo 4 mod 2

assert, type, is, $

La funcion 'assert' sirve para chequear que una expresion es verdadera, si es falsa emite un error, si 'assert' es 'true' no emite ningun mensaje y es silenciosa.

La funcion 'type' devuelve el el tipo concreto de cualquier valor, retorna un tipo 'string'.

La funcion 'is' sirve para chequear si un valor es de un tipo especifico, retorna un tipo 'bool'.

La funcion '$' sirve para convertir cualquier valor a un 'string', de ser posible. Usualmente se llama a '$' como operador "Dolar" o "Stringify", siempre retorna un tipo 'string'.

assert true assert 42 == 42 assert 42 is int assert 9.9 is float assert "x" is string assert true is bool var entero = 42 assert $entero is string assert $42 is string assert $9.9 is string assert $true is string var variable = "ejemplo" echo type(variable) echo type(true) echo type(5)

Variables

Las variables pueden contener distintos valores de cualquier tipo bajo un nombre, idealmente los nombres de las variables deberian ser descriptivos de su contenido. Las variables necesitan ser declaradas usando 'var' y el nombre de la variable, luego de que son declaradas pueden utilizarse asignandole valores del mismo tipo, es decir si una variable tenia enteros solo puede alojar enteros, una variable tenia booleanos solo puede alojar booleanos, etc.

Es posible declarar una variable usando su tipo en lugar de un valor. Es posible asignarle una variable a una variable, copiando una variable en otra. No es posible declarar una variable con el mismo nombre varias veces, por que se perderia el valor de la variable original si se pisan mutuamente. Mas adelante se explicara un uso mas avanzado de las variables. Cuando se crea una variable su valor por defecto es el valor por defecto de su tipo.

  • Tipo 'bool' son valores como 'true' o 'false'.
  • Tipo 'int' son valores como '0', '9', '-1', '-99', '42'.
  • Tipo 'float' son valores como '1.0', '3.14', '-9.5', '5.555'.
  • Tipo 'string' son valores como '"a"', '""', '"OwO"', '"palabras"'.
var numero = 1 # Variable declarada usando un valor. numero = 99 numero = -3 echo numero var texto = "" # Variable declarada usando un valor vacio. texto = "azul" texto = "rojo" echo texto var booleano: bool # Variable declarada usando un tipo. booleano = true echo booleano var copia = numero echo copia # Variable copiada en otra variable. echo numero # var copia = 5 # ERROR: La variable ya existe.

Comparacion

La comparacion booleana sirve para comparar 2 valores o variables, y devolver un booleano 'true' o 'false', dependiendo de los valores comparados.
  • Para comparar si dos valores son iguales se usa '=='.
  • Para comparar si un valor es mayor que otro se usa '>'.
  • Para comparar si un valor es mayor o igual que otro se usa '>='.
  • Para comparar si un valor es menor que otro se usa '<'.
  • Para comparar si un valor es menor o igual que otro se usa '=<'.
var numero = 5 # Puedes probar cambiar este numero entero. echo numero == 5 # Puedes probar cambiar este operador de comparacion por otro.

if, elif, else

Control de flujo con if, elif, else. La indentacion con 2 espacios se usa para marcar grupos de comandos que van juntos, delimita un bloque de codigo.
  • 'if' sirve para ejecutar o no un bloque de codigo dependiendo de una condicion booleana.
  • 'elif' se usa para agrupar varios 'if' juntos de manera tal que solo 1 de los 'elif' se ejecute.
  • 'else' se usa para ejecutar un bloque de codigo si ninguno de los 'if' y 'elif' se ejecuto.
var condicion = false if condicion: echo "condicion es true" else: echo "condicion es false" var numero = 0 if numero == 0: echo "Numero es igual a cero" elif numero == 1: echo "Numero es igual a uno" elif numero == 2: echo "Numero es igual a dos" elif numero == 3: echo "Numero es igual a tres" else: echo "Numero es algun otro valor"

Arrays

Los Arrays son colleciones de valores, valores agrupados en un mismo contenedor, los Arrays son de capacidad y longitud fija, por lo tanto no pueden crecer, no se puede agregar ni quitar mas valores, pero si se pueden modificar los valores existentes, todos los items de un array deben ser del mismo tipo, el array siempre tiene un tipo especifico que puede alojar. El array es construido con '[ ]' y uno o mas valores, es decir por ejemplo '[1]', los items de un Array se separan con comas, es decir por ejemplo '[1, 2, 3]', tambien se puede crear usando el tipo en lugar del valor literal, es decir array[longitud, tipo], por ejemplo 'array[3, int]' para un array de 3 'int', basicamente '[1, 2, 3]' es el mismo tipo que 'array[3, int]', si se trata de 'string' es '["a", "b", "c"]' con tipo 'array[3, string]', si se trata de 'float' es '[1.0, 2.0, 3.0]' con tipo 'array[3, float]', etc.

Array es como seq pero de capacidad estatica.

echo [1, 2, 3] echo ["a", "b", "c"] echo [1.0, 2.0, 3.0] var ejemplo: array[3, int] ejemplo[0] = 1 ejemplo[0] = 42 echo ejemplo[0]

Secuencias

Las secuencias son colleciones de valores, valores agrupados en un mismo contenedor, las secuencias pueden hacerse mas grandes o mas chicas, agregando o quitando valores, todos los items de una collecion deben ser del mismo tipo, la secuencia siempre tiene un tipo especifico que puede alojar, pero puede estar vacia tambien. La secuencia es construida con '@[ ]' y uno o mas valores, es decir por ejemplo '@[1]', los items de una secuencia se separan con comas, es decir por ejemplo '@[1, 2, 3]', tambien se puede crear usando el tipo en lugar del valor literal, por ejemplo 'seq[int]', basicamente '@[1, 2, 3]' es el mismo tipo de secuencia que 'seq[int]', si se trata de 'string' es '@["a", "b", "c"]' con tipo de secuencia que 'seq[string]', si se trata de 'float' es '@[1.0, 2.0, 3.0]' con tipo de secuencia que 'seq[float]', etc.

Seq es como Array pero de capacidad dinamica.

echo @[1, 2, 3] echo @["a", "b", "c"] echo @[1.0, 2.0, 3.0] var secuencia0: seq[int] var secuancia1: seq[bool] var secuencia2: seq[float] var secuencia3 = @[true, false, true]

Operaciones con Secuencias

Los items dentro de la secuencia tienen un numero entero de indice unico, el indice de las secuencias comienza por '0' para el primer item, hasta el ultimo item de la secuencia. No existen los indices negativos.

Los items de una secuencia se pueden obtener por indice usando '[]', es decir secuencia[indice], por ejemplo 'secuencia[3]', que devuelve el item en el indice '3'. Para obtener un item por indice al inverso (indice invertido) es decir desde el ultimo hacia el primero, se puede usar secuencia[^indice], por ejemplo 'secuencia[^1]', que devuelve el ultimo item en el ultimo indice, por ejemplo 'secuencia[^2]', que devuelve el anteultimo item en el anteultimo indice, etc.

Para asignar un valor a un item de una secuencia, es decir modificar un item de la secuencia, se pueden asignar por indice usando '[]=', es decir secuencia[indice] = valor, por ejemplo 'secuencia[3] = 42', que asigna '42' al item en el indice '3'.

Para saber la longitud total de una secuencia se puede usar 'len', es decir 'len(secuencia)' que devuelve un numero entero que es la longitud.

Para agregar items a una secuencia se usa 'add', es decir 'secuencia.add(nuevoItem)'.

Para borrar items de una secuencia se usa 'delete', de la forma 'secuencia.delete(indice)', es decir 'secuencia.delete(3)' para borrar el item en el indice '3'.

var secuencia: seq[int] echo secuencia echo len(secuencia) secuencia.add(1) secuencia.add(2) echo len(secuencia) echo secuencia[1] secuencia.add(3) secuencia[1] = 9 secuencia.delete(1) echo secuencia # secuencia.add(2.0) # ERROR: Es una secuencia de 'int' y '2.0' es 'float'. # secuencia[9] = 5 # ERROR: No existe el indice '9' en la secuencia. # echo secuencia[-1] # ERROR: No existen los indices negativos.

Operaciones con Secuencias 2

El operador '@' puede convertir un Array en Seq. Array y Seq usan una interface de uso similar.

Slicing

Partir una secuencia en otra secuencia mas chica se llama "Slicing", se realiza con el operador '[]' y '..' con rango de numeros enteros, es decir de la forma secuencia[inicio..fin], por ejemplo 'secuencia[0..3]' para una secuencia desde indice '0' hasta indice '3', tambien puede usarse Indices Reversos, con el operador '^' y el numero pegado, es decir secuencia[^inicio .. ^fin], por ejemplo 'secuencia[^2 .. ^1]' para una secuencia desde el ultimo indice hasta el anteultimo indice.

var variable = [1, 2, 3, 4] assert variable[0 .. 0] == @[1] assert variable[0 .. 1] == @[1, 2] assert variable[0 .. 3] == @[1, 2, 3, 4] assert variable[^2 .. ^1] == @[3, 4] assert ^1 is BackwardsIndex # ^1 es un Indice en Reverso. var arra = [1, 2, 3] var seqArra = @arra # 'array[int]' convertido en 'seq[int]' usando '@'. seqArra.add(4) echo seqArra
